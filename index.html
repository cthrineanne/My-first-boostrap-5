Boostrap

My takeaways

Ang Boostrap is open source front-end development na ginagamit ng mga developer or programmer para mag create ng isang website or web application. Ginagamit din yung boostrap in a easiest way to have a responsive design. Mas magandang gawin to lalo na sa mga beginners pa lang kasi isa sya sa pinaka madaling gawin kumbaga pinag combine na yung HTML and css when we create a boostrap unlike kapag gagawa tayo ng HTML and CSS magkaibang page pa sila pag gagawaan natin and kailangan naka organize talaga but sa boostrap we can create a responsive webpage without creating other page kumbaga sa boostrap nandito na yung HTML and CSS nya nakapaloob lang sa iisang page. But the difference lang ng Boostrap may limitations lalo na sa pag cecreate ng mga design unlike sa css lahat pwede mo gawin saknya. Kase yun naman ang role talaga ng CSS is makapag boost ng iba't ibang design. But if you want a simple webpage na mayroon nang responsive design pwede natin gawin in a easiest way ang Boostrap.
